# 动态规划解法专题



## 股票交易获利问题

参考:https://segmentfault.com/a/1190000006672807#articleHeader3



### 只买卖一次

已知一段时间内股价有涨跌，例如 {3,6,1,4,7,3,8,8,3,5}.求最大获利是多少。不能买空卖空。

例如，上例应该是第3天买，第7天卖获利最大，获利是7元。

思路: 

这个比较简单, o(n)的解法只需在遍历每个元素的时候,记住到当前为止的最低价格, 以及在各个时刻卖出, 能获得的最大收益即可.



### 简单买卖：可以买卖无穷多次
