给目录添加远程代码仓库, 关联码云:

```
cd 到git目录
git init
git remote add gitee git@gitee.com:hongbozhang/目录名.git
git remote -v   //查看远程库信息:
```



或者新建一个空目录:

mkdir  reponame

cd reponame

git init

git add file.xxx

git commit -m "your comments"





推送到码云:

1. 在码云上建立相同目录名的目录

2. 推送:

   ```
   git push gitee master
   ```

   

这样一来，我们的本地库就可以同时与多个远程库互相同步： 

```
┌─────────┐ ┌─────────┐
│ GitHub  │ │  Gitee  │
└─────────┘ └─────────┘
     ▲           ▲
     └─────┬─────┘
           │
    ┌─────────────┐
    │ Local Repo  │
    └─────────────┘
```
